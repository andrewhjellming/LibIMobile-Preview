## What is LibIMobile?

LibIMobile replaces 30+ scattered legacy iOS tools with a single, modular C++23 library ‚ÄĒ **235 modules across 8 tiers**. It covers everything from low-level USB transport and lockdown authentication through backup/restore, jailbreak, hardware repair, and forensics, all under one unified API.

The source code is private. This repository is a public preview showing what's being built.

## Highlights

- **USB + Wi-Fi device detection** ‚ÄĒ WinUSB / libusb, usbmuxd multiplexer, USB-NCM, mDNS discovery
- **Full lockdown stack** ‚ÄĒ TLS pairing, trust dialog, service start, get/set values
- **iOS 17+ CoreDevice** ‚ÄĒ QUIC tunnel, XPC binary framing, RemoteServiceDiscovery
- **Backup & Restore** ‚ÄĒ full encrypted backup pipeline, TSS/APTicket, NOR flash, IPSW
- **Instruments / DTX** ‚ÄĒ CPU, memory, energy, GPU, network profilers via DTX wire protocol
- **XCTest / XCUITest** ‚ÄĒ gesture injection, element query, screenshot comparison
- **Jailbreak (checkm8)** ‚ÄĒ DFU/iBoot chain, kernel patch finder, ramdisk injection, keychain dump
- **Forensics** ‚ÄĒ artifact extraction, SQLite carving, chain-of-custody, write-block
- **Hardware Repair** ‚ÄĒ PCB boardview (ODB++/GenCAD), netlist graph, 2D canvas, 3D teardown
- **Windows GUI** ‚ÄĒ Qt6 device panel with 3D model viewer (Qt Quick 3D / glTF), screen mirror, instruments, MDM, filesystem browser
